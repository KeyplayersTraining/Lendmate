Platform Overview

What Filogix Expert is, what it does, and how to navigate it as a VA.

What is Filogix Expert
Primary Purpose
Filogix Expert is a mortgage origination platform used by Canadian mortgage brokers. VAs use it to create and manage mortgage applications, enter borrower data, and submit files directly to lenders and insurers (CMHC, Sagen, Canada Guaranty).
Most common platform in Canadian brokerage
Application Types
Filogix supports three transaction types: Purchase (buying a new property), Refinance (changing terms on an existing mortgage), and Switch/Transfer (moving to a new lender without borrowing more). Each type has its own required fields.
Know which type before opening a file
Key Advantages for VAs
Filogix automatically calculates qualifying income based on the income types you enter. It flags errors and warnings before submission so you can catch problems early. It also connects directly to lender systems — no manual email required for integrated lenders.
Always check the Errors/Warnings panel before submitting
How to Log In and Navigate
1
Log In Securely
Always use the Log Out button when finished — do not just close the browser. Closing without logging out can leave the session active, which is a compliance and security risk.
2
Find an Existing File
Use the Find/Search function to locate existing applications. Never scroll through the full list manually — use the search by borrower name, file number, or date.
3
Access Help & Training Resources
Training guides and user documentation are available under the Help section within Filogix. Check here first if you are unsure how a field works before escalating to the broker.

Client Information

Where and how to enter borrower and co-applicant details in Filogix.

Borrower Details Section
Where to Enter Personal Information
All borrower personal data goes into the Borrower Details section — not the Mortgage Request tab, not the Notes section. Minimum required to create a new applicant: Full name and date of birth. SIN, address, and employment are filled in next.
Adding a Co-Applicant
Use the "Add Applicant" button inside the existing file — do not create a separate application. All borrowers on the same mortgage must be inside the same Filogix file.
Never create duplicate files for co-applicants
Required Minimum Fields
To save and move forward, you need at minimum: legal name (as on government ID) and date of birth. Everything else — SIN, address, marital status — follows in the same section before submission.
Match name exactly to government ID

Employment & Income

How to enter income data correctly so Filogix calculates qualifying income accurately.

Employment/Income Section
Where to Enter It
All employment and income data is entered in the Employment/Income section — not under Assets, not under the Mortgage Request tab. Each income source gets its own entry.
Valid Income Types
Accepted: Salary, Hourly, Commission, Self-Employment, Rental, Pension/CPP, and other CRA-recognized income. Cryptocurrency staking returns are not an accepted income type in Filogix — flag to broker if client lists this.
Crypto income = flag to broker
How Qualifying Income is Calculated
Filogix calculates total qualifying income automatically based on the income types and amounts you enter. You do not calculate it manually. Your job is to enter the data accurately — the system does the math. Always cross-check the calculated amount against the income documents provided.
Your job: accurate data entry. Filogix does the calculation.

Assets & Liabilities

Understanding what counts as an asset vs. a liability, and how to update each.

Assets vs. Liabilities — Know the Difference
Assets (What the Borrower Owns)
Examples: savings account balances, RRSP/TFSA/investment accounts, vehicle value, real estate equity. These are things that have positive monetary value that the borrower owns outright or partially owns.
Down payment funds are an asset
Liabilities (What the Borrower Owes)
Examples: mortgage loans, car loans, student loans, credit card balances, lines of credit. These reduce the borrower's net worth and affect their debt service ratios. Enter all active liabilities — do not omit any.
Never omit a liability — it affects qualification
Updating an Existing Liability
To edit an existing liability entry, double-click the liability to open its fields for editing. Do not delete the entire record and re-enter — this creates duplicate entries and compliance issues. Update the relevant field, save, and move on.
Double-click to edit — never delete and re-enter

Property & Application Details

Where property data lives in Filogix and how to verify the file before submission.

Property Details Tab
Where Property Info Goes
All property-related data (address, type, value, occupancy) is entered in the Property Details tab. Loan amounts, interest rates, amortization, and term go in the Mortgage Request section. These are two separate tabs — do not mix them up.
Property Details tab ≠ Mortgage Request section
Verifying Required Fields
Before submitting any file, check the Errors/Warnings panel. This panel flags every incomplete or incorrect field. Do not rely on the broker to catch missing fields — run this check yourself as the last step before handing the file over.
Errors/Warnings panel = your pre-submission checklist

Documents, Compliance & Lender Submission

How to upload documents, manage consent, record notes, and submit to lenders.

Document Management
Uploading Documents
All supporting documents are uploaded through the Documents tab inside the file — not in Assets, not in the Mortgage Request section. Consent forms must be uploaded before a credit check can be run.
Why Consent is Required
The consent form gives the lender legal permission to run the borrower's credit bureau. Without it, running credit is a privacy law violation. This is not optional — collect consent before any credit pull.
No consent = no credit pull. Period.
Lender Submission
1
Select Lenders
Use the Lender List checkbox menu to select which lenders to submit to. Filogix does not auto-select lenders — the broker decides, the VA executes. Never submit to a lender without broker confirmation.
2
Review Before Submitting
Check: all data is accurate, all documents are uploaded, no Errors/Warnings outstanding. Do not submit a file with known issues.
3
Check Lender Responses
After submission, monitor the Lender Conditions/Messages section — not your email inbox. Lender responses, conditions, and requests for additional information all come through this section.
Compliance Notes
Why Notes Matter
All compliance notes go in the Notes/Compliance section. These notes create a record of client conversations, decisions, and instructions. In the event of a compliance audit, missing notes = missing evidence. Log every significant action or client instruction in notes immediately.
